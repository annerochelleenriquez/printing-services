# PrintCraft ERP

## Project Overview

PrintCraft ERP is a modern print shop management system.

The application has two main parts:

1. Customer Website
2. Admin Dashboard

The customer website allows users to:

- Browse products
- Upload artwork
- Configure print options
- View live pricing
- Add items to cart
- Checkout
- Track orders

The admin dashboard allows staff to:

- Manage orders
- Process production
- Manage inventory
- Configure pricing rules
- Use a POS for walk-in customers
- View reports

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Prisma
- PostgreSQL
- shadcn/ui
- Auth.js
- Cloudinary
- Stripe (future)

## Design Requirements

- Keep the existing homepage design.
- Modern, premium, and responsive UI.
- Mobile-friendly.
- Use reusable components.

## Goal

Build a production-ready print shop ERP system that combines:

- Online ordering
- Print pricing calculator
- POS
- Inventory management
- Production workflow
- Reporting
